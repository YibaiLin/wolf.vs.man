## 狼人杀——上帝助手

这是一个网页小程序，学习 jQuery 时用来练习的。

### “上帝助手”的定位

我把此应用定位为“上帝助手”，即它可以用来充当一副狼人杀牌，同时它可以帮助“上帝”更容易地记住游戏的过程。

### 版本更新

2017.08.21 对该应用进行了 UI 的更新、代码的精简、增加了更多注释，全然一新。
